概述

Entry.css提供一种可配置的、更适合阅读的中文文章样式，满足一般的博客文章展示需求。它使用了Less css作为开发工具。同时推荐使用markdown来书写文章，然后转换成HTML。

块级元素

段落和换行

段落的HTML只使用<div>和<p>。它们的行高为1.62em，行首缩近两个中文字，上下之间有1em的间距。如果处于默认的左排版样式，则会距左内容框有1em的距离。右排版样式则是距右内容框1em距离，并文字向右对齐。居中排版则没有边距。

换行也可以使用<br>标签实现。

标题

文章标题

文章标题是整个HTML中的特殊部分，它使用entry-header为class的<div>标签来包裹。其中包含使用<h1>和使用.entry-meta两个部分。前者是标题，后者是Meta信息。

<div class="entry-header">
    <h1>标题</h1>
    <div class="entry-meta">
        作者：<a href="http://mzhou.me/">mzhou</a>
    </div>
</div>

文章内部标题

文章内部标题共分为四种，使用<h1>标签作为一级标题，<h2>作为二级标题，<h3>标签作为三级标题，<h4>、<h5>或<h6>标签作为四级标题。

一级标题字体为2em，二级为1.5em，其余的样式它们都一致。三级标题字体为1.2em，并且有一个icon（可配置）。四级标题会仅仅的贴着接下来的段落，字体为1em且加粗。

列表

列表样式分为有序列表<ol>和无序列表<ul>两种，这两种可以相互嵌套。但是没有对列表样式做过多的修改，还是使用浏览器默认样式。

区块引用

区块引用只使用<blockquotes>标签。它们的样式使用了两个Unicode实体图标，这两个icon可以自定义。区块引用的样例如下：

人生不过是一个行走的影子，一个在舞台上指手划脚的笨拙的怜人，登场片刻，便在无声无息中悄然退去，这是一个愚人所讲的故事，充满了喧哗和骚动，却一无所指。 麦克白

代码区块

代码区块样式使用<pre>标签作为容器，且内部可选地嵌入<code>标签。推荐使用prism实现代码高亮。

HTML例如：

<link rel="stylesheet" href="prism.css" />
<pre><code>
console.log('Test');
</code></pre>
<script src="prism.css"></script>
<blockquote>与<pre>的内容部分一样，都距离内容框4em。

分割线

分割线样式只能使用<hr>标签。分割线样式在边缘部分使用CSS3的linear-gradient样式做了渐变处理，所以部分浏览器会不支持。

行内元素

链接

连接的样式可以自定义链接颜色。对于href属性值以javascript:开头的a标签样式比较特殊。例如：Javascript连接和有onclick属性的连接和普通链接。

强调

强调的样式可以使用<strong>、<u>、<b>或<em>标签，<strong>表示语义与样式双重强调，后两者表示样式上的强调。<b>标签是加粗的样式，<em>标签则有虚线下划线，相邻的<em>之间会有0.125em的间隔。<u>标签的样式是直线下划线，相邻之间也有0.125em的间隔

代码

代码的行内样式使用<code>标签，默认的颜色样式来自著名的前端库Bootstrap。你也可以自定义颜色。

其他行内样式

  1. 书名号使用<cite>标签，例如：

    冰与火之歌是一部很棒的魔幻小说！
    Hamlet也是一部很棒的小说！

  2. 缩写使用<abbr>标签，例如：

    js 的全称是javascript，鼠标放上去看看title值。

  3. 定义使用<dfn>标签，例如：

    HTML 是超文本标记语言

  4. 标记使用<mark>标签，例如：

    在这句话里面把这几个字高亮下吧～

  5. 行内引用使用<q>标签，例如：

    在这句话里面把这几个字引用下吧～

  6. 旁注使用<small>标签，例如：

    单人间 399 元 含早餐，不含税

  7. 上下标使用<sup>标签作下标、<sub>标签作上标，例如：

    105 与 a12

配置及使用

Entry.css是基于less css开发的，故提供了配色和部分icon的配置项。首先安装nodejs和npm然后运行npm install less -g安装less css。配置文件在src目录下的config.less文件中。可配置的样式选项及介绍如下：

@content-width: 600px;  // 内容区的宽度
@font-size: 14px;       // 基础文字大小
@line-height: 1.62;     // 行高

@head-color: #83A1D5;   // 文章标题背景色

@word-color: #666;              // 文字默认颜色
@word-highlight-color: #FFF;    // 文章标题在背景高亮时候文字默认的颜色
@word-back-color: #83A1D5;      // 文章标题在背景高亮时候文字默认的背景颜色
@word-highlight-color-2: #FFF;  // 一级标题在背景高亮时候文字默认的颜色
@word-back-color-2: #96B3D3;    // 一级标题在背景高亮时候文字默认的背景颜色

@link-color: #069;              // 普通链接颜色
@link-highlight-color: #FFF;    // 普通链接在背景高亮时候的颜色
@link-visited-color: #551A8B;   // 普通链接在访问过后的颜色
@link-js-highlight-color: #FFF; // 执行js的链接颜色
@link-js-back-color: #96B3D3;   // 执行js的链接背景色

@blockquote-quot-color: #CCC;       // 引用icon的颜色
@blockquote-before-icon: '\275D';   // 前一个引用icon
@blockquote-after-icon: '\275E';    // 后一个引用icon

@h3-icon: '\27BD';      // h3的icon

// 从bootstrap拷贝的颜色
@code-color: #D14;              // code的文字颜色
@code-back-color: #F7F7F9;      // code的背景色
@code-border-color: #E1E1E8;    // code的边框颜色

@hr-color: #888;                // hr的颜色

// 从bootstrap拷贝的颜色
@table-border-color: #DDD;              // table边框颜色
@table-inner-border-color: #DDD;        // table内边框颜色
@table-highlight-back-color: #F9F9F9;   // table背景高亮颜色

@img-border-color: #CCC;    // 图标边框颜色

@mark-back-color: yellow;   // mark的背景色

基于less可以自定义文章标题和文章内容区域的CSS选择符。src目录下的wordpress.less是针对如下HTML结构的：

<article class="post">
    <header class="entry-header">
        <!-- 文章标题 -->;
    </header>
    <!-- 内容 -->
</article>

这种HTML结构与一般的Wordpress博客结构类似。wordpress.less文件如下：

@import "entry.less";
.post {
    .ext-entry-content;
}
.post .entry-header {
    .ext-entry-header;
}
.post .entry-header .entry-meta {
    .ext-entry-meta;
}

然后运行命令lessc wordpress.less > wordpress.css，生成wordpress.css文件就可以被用到你的博客样式中。你可以先修改default.less，然后打开index.html查看效果～如果你设计了不错的配色，欢迎pull request。

也可以使用默认提供的几种结构，它们都在bin目录下。

  • wordpress.css专用于一些基于wordpress博客。它适用的HTML结构如上所说。
  • markdown.css用于markdown生成的页面，不过需要在body上加class="markdown"，如果需要文章标题样式也需要自己添加<div class="markdown-header">标签。这里有demo。它适用的HTML结构如下： 
    <body class="markdown">
    <div class="markdown-header">
        <!-- 文章标题 -->;
    </div>
    <!-- 内容 -->
</body>
  • puretag.css用于一般情况。它适用的是纯html结构： 
    <body>
    <header>
        <!-- 文章标题 -->;
    </header>
    <!-- 内容 -->
</body>
